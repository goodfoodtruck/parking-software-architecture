# 003. Technologies utilisées pour l'application client
## Status
Proposed

## Contexte:
L'application client est fortement dépendante des règles métier, du typage
est donc préférable.
Plusieurs options sont possibles pour développer une application client:
- Python
- Java
- Javascript
- Flutter
- Etc

L'équipe de développement maîtrise principalement Typescript et React.

## Decision:
Dans ce contexte, la meilleure option serait de développer l'application web avec
React et Typescript.

## Consequences:
#### Positive:
Le développement des features sera plus rapide et le code
suivra les bonnes pratiques de développement avec ces technologies.
#### Negative:
Performances plus coûteuses (négligeable)
